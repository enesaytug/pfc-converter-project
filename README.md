- Project Context: This work was conducted as a term project for the ELK453E (Industrial Applications of Power Electronics) course.

- Development Methodology: The design and implementation of this website were assisted by advanced Large Language Models, including Claude 3.6 Sonnet, Google Antigravity (Gemini 3.1 Pro), and ChatGPT 5.5.

- System Performance & Stability: While the system demonstrates overall stability—evidenced by a steady 400V DC output voltage, adequate phase margins, and generally low Total Harmonic Distortion (THD)—certain performance limitations remain. Specifically, the input current THD exhibits fluctuations, ranging from an optimal 4.4% up to 12%. Furthermore, the implemented controller did not fully reach the theoretically targeted crossover frequencies.

- Future Work: For future improvements, the system's dynamic response can be optimized by fine-tuning the Proportional-Integral (PI) controller parameters and re-evaluating the frequency response via updated Bode plots to achieve the desired crossover frequencies.
